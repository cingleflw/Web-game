# Web Game — Point & Click Adventure

Браузерная point-and-click игра с 5 уровнями, инвентарём и загадками.


## Установка и запуск

```bash
# 1. Установите зависимости
pip install flask

# 2. Запустите сервер
python app.py

# 3. Откройте в браузере
http://localhost:5000
```

## Управление

| Клавиша | Действие |
|-----------|----------|
| **←** (стрелка влево) | Предыдущий уровень |
| **→** (стрелка вправо) | Следующий уровень |
| **Клик по предмету** | Взаимодействие |
| **Клик по слоту инвентаря** | Выбрать предмет для использования |
| **Цифры 0–9** | Ввод кода в кодовый замок |
| **Enter** | Подтвердить код |
| **Escape / Backspace** | Закрыть замок |

## Механика игры

### Общие правила

- **Кот** (`cat.png`) присутствует на каждом уровне. Нельзя взять, нельзя использовать.
- **Предметы в инвентаре**: клик по слоту выделяет предмет (золотая рамка). Клик по объекту на уровне с выбранным предметом — использование.


## Технологии

- **Backend**: Python + Flask
- **Frontend**: JavaScript (ES6), CSS3
- **Изображения**: PNG с прозрачностью, JPG для фонов
- **Разрешение уровней**: 734×411 px
